## 3D Portfolio
This is my portfolio built using React and Vite.

## Tech Stack
* React.js
* Email JS
* Vite
* Tailwind CSS
* i18next

## Features
* Responsive design using Tailwind CSS.
* Multi-language support via i18next.
* Contact form integrated with EmailJS.
